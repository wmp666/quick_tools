# 快捷启动工具
## 功能
为各种小工具提供快捷启动入口
## 使用方法
在获取了基于`develop_tool.jar`开发的jar包工具后，将工具拖入软件的安装目录下的`app/tools`文件夹，按下刷新
## 下载链接
[开发工具包](https://github.com/wmp666/quick_tools/releases/tag/0.2)
[官方工具库](https://github.com/wmp666/quick_tools/releases/tag/0.1)
